# ai-basics
1. What is AI Agent
  - AI agent autonomously performs tasks with varying complexity
   Input taks-> Planning -> Action -> Feedback -> Implement
3. What AI Generative AI
   - GA** plan and execute actions** autonomously by following a structured loop of goal setting, information gathering, planning, and action execution - all based on their understanding of the environment and objectives
4. Role of AI agent
  -  Generative AI is revolutionizing customer inquiry handling by enabling businesses to automate interactions, and personalize responses, and enhance overall service efficiency.
  -  AI agents **gather** necessary data, **plan action** sequences using reasoning models, and **execute tasks** through available tools or APIs
  -  Agents **evaluate outcomes after each step**, loop or branch tasks as needed, **retry failures**, **request clarifications**, or escalate issues to humans, ensuring robust, accurate, and goal-aligned customer inquiry resolution.
